Introductie

Deze pagina geeft toegang tot voorbeelden uit het boek. De voorbeelden bestaan meestal uit korte stukjes HTML, CSS of beide. Schakel de bewerkmodus in en pas de code aan. Het resultaat ziet u direct in de weergave boven de code. U kunt code veranderen, verwijderen en toevoegen, maar u kunt niet plakken.

Met de kopieerknop maakt u op elk gewenst moment een kopie van de code naar het klembord voor gebruik in uw editor.

Links uit het boek

U vindt hier ook alle links uit het boek, verzameld per hoofdstuk en in dezelfde volgorde als in het boek.

Complete pagina's

Sommige technieken staan op een eigen pagina. Dat zijn:

  • Een compleet aanmeldingsformulier met verwerking op de server
  • CSS Grid Layout (voorbeeld voorpagina nos.nl). De complete code voor deze pagina kunt u downloaden van GitHub.

Fouten en verbeteringen

Op pagina 227 staat bij oplossing 2 dat de specificiteit 21 is. Dat is fout. De juiste waarde is 11. Een attribuutselector en een klasse tellen ieder voor 10 en een element voor 1. Ook bij oplossing 3 is de uitkomst niet 21 maar 11.

In de voorbeelden 8.1 en 8.2 zat een fout waardoor deze voorbeelden niet het goede resultaat lieten zien. Die fout is hersteld, waardoor spelen met de cascade layers nu wel het bedoelde resultaat geeft.
