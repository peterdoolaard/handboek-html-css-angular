Baseline: kan ik feature xyz gebruiken in productie?

Wanneer is een nieuwe feature van HTML, CSS of JavaScript bruikbaar op een live-site? Het antwoord op deze vraag is iets makkelijker geworden met de invoering van het begrip baseline.

Wat is de baseline?

Baseline is een ‘streep in het zand’ die aangeeft of een feature klaar is voor algemeen gebruik. Dat was in de eerste versie het moment waarop de feature is ingebouwd in de belangrijkste browsers. Maar dat betekent niet dat deze daadwerkelijk beschikbaar is voor alle gebruikers en dat helpt webdevelopers nauwelijks verder. Daarom is besloten twee strepen te trekken.

In de nieuwe definitie zijn er twee baselines:

  • Nieuw beschikbaar. Het is interoperabel, wat wil zeggen dat de belangrijkste browsers het ondersteunen.
  • Breed beschikbaar. De feature kan in het algemeen veilig worden gebruikt. Dit punt wordt bereikt 30 maanden nadat de feature nieuw beschikbaar was.

De baseline wordt gebaseerd op beschikbaarheid in Chrome, Edge, Firefox en Safari.

Hoe hard is de baseline?

Een streep in het zand is geen harde grens. 30 maanden is een ruime, veilige richtlijn gebaseerd op wereldwijd gebruik.

Een webapplicatie voor een specifieke groep waarvan bekend is dat die altijd een bijgewerkte browser gebruikt, kan vertrouwen op ‘nieuw beschikbaar’. Of zelfs nog eerder als de specifieke browser het al ondersteunt.

In alle ander gevallen is voorzichtigheid verstandig. In principe wilt u geen gebruikers uitsluiten door niet-ondersteunde features toe te passen. We kennen wilswaar de evergreen browser die veel updates krijgt en die ook klaarzet, maar een gebruiker moet altijd nog het laatste zetje geven. Er zijn mensen die updates uitstellen of bewust vermijden, die niet kunnen updaten omdat er iets fout zit dat ze niet kunnen oplossen, of die oudere hardware gebruiken waarop de nieuwste browserversie niet wordt geïnstalleerd (denk iPad of iPhone).

Waar vindt u de baseline?

Het Mozilla Developer Network, beter bekend als MDN, publiceert de baseline in de naslag voor HTML, CSS en JavaScript. Op dezelfde pagina is ook te vinden vanaf welke browserversie de feature beschikbaar is.

Misschien mist u nu de website caniuse.com. Deze publiceert de baseline ook. Hoewel Can I Use ook zelf nieuwe features test, baseert het zich vaak op data van MDN. Daarmee is MDN de beste bron, maar u kunt vanwege het gemakkelijk zoeken ook Can I Use gebruiken.
